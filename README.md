# MetroCity College Database Documentation Project

## Project Overview

The MetroCity College Database Documentation Project is a MySQL-based database developed to manage student records, academic departments,
course registrations, and examination results. This project focuses on database documentation, collaboration, and version control 
using professional database management tools.

---

## Project Objectives

The objectives of this project are to:

- Create a standard data dictionary using Microsoft Excel.
- Generate professional database documentation using dbForge Studio.
- Manage database project files using Git and GitHub.
- Create a professional README.md file.
- Upload all project files to a GitHub repository.
- Demonstrate database documentation and collaboration practices.

---

## Database Description

The database manages academic information for MetroCity College. It stores student details, department information, course records, 
student enrollments, and examination results. The database uses primary keys and foreign keys to maintain data integrity and
relationships between tables.

---

## Database Tables

- Departments
- Students
- Courses
- Enrollments
- Exam_Results

---

## Main Features

- Manage department information.
- Store student records.
- Manage course details.
- Record student enrollments.
- Store examination results, scores, and grades.
- Maintain relationships using primary and foreign keys.
- Support database documentation and collaboration.
